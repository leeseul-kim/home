			***숫자야구***

- 숫자 4자리를 받을 배열 / ArrayList 준비 -> 클래스? 
- 각 자릿수에 들어갈 숫자는 0~9까지 랜덤하게 중복없이 추출
- 추출된 숫자는 사용자에게 보이지 않아야 함
- 사용자에게 숫자를 입력받기 
	이때, 숫자만 입력받아야한다. 
	+) 4자리 이상이거나, 중복된 수가 입력되면 재입력 받도록 하기 
		이때, 중복된 수가 입력되는 경우에 대한 예외 클래스 만들어주기 
	+) 4자리 미만의 숫자가 입력되면 재입력받게 하기 
		+) 1000미만으로 조건문을 걸면 안됨 
- 사용자가 입력한 숫자와 추출한 숫자를 비교
- 숫자만 맞으면 ball, 자리까지 맞으면 strike, 아무 숫자도 포함되지 않는다면 out을 반환
- i가 0일때, j가 0~3번까지 다 돌면서 숫자가 겹치는지 확인해주어야한다.
	단, 이때 동일한 자리까지 맞는다면 스트라이크, 숫자만 맞는다면 볼을 출력해준다.





- 사용자가 얼마만에 맞췄는지도 체크해주기


- 가능다면, 클래스는 
				1. 메인클래스
				2. 컨트롤클래스
				3. 인풋클래스
				4. 아웃풋클래스
	로 나누어서 제작해보기.
